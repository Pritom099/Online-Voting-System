# 🗳️ Online Voting System (C++)

A simple console-based Online Voting System built using C++ with file handling and OOP concepts.

---

## 🚀 Features

- Voter Registration & Login
- Age Verification (18+ only)
- One Person One Vote system
- Vote stored using file handling
- Admin Panel:
  - View total votes
  - Show winner
  - View all voters

---

## 🛠️ Technologies Used

- C++
- File Handling (`fstream`)
- Object-Oriented Programming (OOP)

---

## 📁 Required Files

Program automatically creates/uses:

- `voters.txt` → Stores voter information
- `votes.txt` → Stores vote counts
